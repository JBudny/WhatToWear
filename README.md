# WhatToWear

You don't know what to wear?
This application will help you to dress up according to the weather.

## Installation

Type in cmd:
```
npm install
```
or
```
yarn install
```

## Running

Type in cmd:
```
npm start
```

Address and port of the started server:
```
localhost:3000
```
